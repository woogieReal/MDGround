# MDGround
> 마크다운 언어를 사용한 SNS

​
일상생활과 프로그래밍 관련 포스트를 마크다운 언어를 사용하여 게시할 수 있는 개발자들을 위한 SNS 서비스  

​
## 프로젝트 목표
 * 클라우드 컴퓨팅(AWS)를 사용하여 배포
 * Bootstrap을 사용하여 PC와 모바일에서 동시에 이용할 수 있는 반응형 웹사이트 구축
